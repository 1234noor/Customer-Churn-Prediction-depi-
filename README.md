# Customer Churn Prediction

This project predicts whether a bank customer will churn using Machine Learning.

## Technologies
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy

## Model
Gradient Boosting Classifier

## Features
- Credit Score
- Age
- Tenure
- Balance
- Products Number
- Credit Card
- Active Member
- Estimated Salary
- Country
- Gender
